# MedScanner AI

An intelligent medicine analysis platform built with Next.js and Google Gemini.

## Features
- **AI Medicine Identification**: Scan any medicine strip or bottle to get instant, detailed information.
- **Cross-Platform**: Optimized for both desktop (file upload) and mobile (camera capture).
- **Clinical Insights**: Provides salt composition, uses, dosage forms, benefits, and critical warnings.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **AI**: Google Gemini Flash 1.5
- **Styling**: Tailwind CSS 4 + Framer Motion
- **Storage/Logs**: Google Drive & Google Sheets API

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables in `.env.local`:
   ```env
   GEMINI_API_KEY=your_api_key
   GOOGLE_SHEETS_ID=your_sheet_id
   GOOGLE_SERVICE_ACCOUNT_EMAIL=your_email
   GOOGLE_PRIVATE_KEY=your_private_key
   DRIVE_FOLDER_ID=your_folder_id
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
